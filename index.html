Firefox Vim Keybindings

  • r: Reload page
  • f: Search link
  • Ctrl + Tab: Move between tabs
  • b, o: Search
  • y: Copy the current URL
  • p: Go to the previous page
  • ?: Show datasheet
  • yt: Duplicate current tab
  • X: Restore closed tab
  • T: Search through current tab
  • gi: Focus the first input box
  • L: Go forward in the history
  • H: Go back in the history
